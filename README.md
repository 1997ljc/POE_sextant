# poe_sextant
流放之路，自动洗罗盘，还在开发中。 
  
  

目前使用时，需要在背包的第一列放置六分仪，第二列放置测绘罗盘，  
一次最多洗50个，可以通过代码来修改自己想要过滤掉的低价值罗盘，  
运行时，出现的“鼠标放置到XX处”，此时敲击空格即可记录下位置，  

运行过程中如果想要停止，疯狂敲击空格即可！！！！
